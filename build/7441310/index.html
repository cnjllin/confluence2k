一、查看设备类型

1. 选择[数据存储]模块；

2. 选择[设备类型]；

3. 设备类型查看列表，包含设备类型ID、设备类型名称等概要信息；

4. 点击某个设备类型的ID，可以进一步查看该设备类型的详细信息；

5. 添加设备类型；

二、查看设备类型详细信息

1. 下钻到设备类型包含的设备列表；

2. 查看设备类型某个传感器上的聚合信息；

3.  编辑设备类型；

三、编辑设备类型

1. 编辑设备类型名称；

2. 编辑描述信息；

3. 编辑标签；

4. 编辑属性；

5. 保存或者取消修改操作；

四、查看设备

1. 选择[数据存储]模块；

2. 选择[设备]；

3. 设备查看列表，包含设备ID、设备名称、设备类型ID等概要信息；

4. 点击设备ID，进一步查看设备详细信息；

5. 添加设备按钮；

五、查看设备详细信息

1. 设备基本信息；

2. 设备中传感器的聚合信息；

3. 编辑设备按钮；

六、查看动态数据-1

1. 选择[数据存储] --〉[设备]，在页面下方有传感器列表；

2. 点击需要查看的传感器ID，进一步查看该传感器对应的动态数据；

七、查看动态数据-2

1. 传感器信息；

2. 设备类型信息；

3. 设备对应的compoundId信息；

4. 该传感器的聚合信息；

5. 该传感器动态数据的时序图；
