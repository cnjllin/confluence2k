以计算每日平均风速和平均发电功率为例，演示如何使用PAS进行数据分析。

一、浏览分析项目

在左侧导航栏点击“分析项目”，可以浏览已有的分析项目。

二、创建分析项目

点击“创建项目”按钮，进入分析项目创建向导，向导分为三个步骤。

步骤1 填写基本信息

(1) 输入项目名称：day-windspeed-production

(2) 选择分析类型：PreProcessing

(3) 输入项目描述信息：（可以为空）

(4) 上传分析程序：daySummaryT.R

参考

library(plyr) 
# mainfunction("E:/job/PAS/KMX-demo/handbook.csv", "E:/job/PAS/KMX-demo/1.csv")

  DFrame <- DFrame[order(DFrame$turbineId, DFrame$ts), ] #sort by time 
   
  DFrame <- transform(DFrame, date = substr(DFrame$ts,1,10)) 
 
  outFrame <- ddply(DFrame, .(turbineId, date), summarize,  
                avgWindSpeed = round(mean(windSpeed), 2), 
                dayProduction = round(sum(powerActive)/120, 2)) 
  ##calculate the average wind speed of every single day  
  
  fileName <- "daySummary.csv" #specify the filename 
  fileDirName <- paste0(dirname(output), "/", fileName) #get the full path 
  write.csv(outFrame, file = fileDirName, row.names = F, quote = F) #write the csv 
  return(fileDirName) 
}

 

(5) 填写算法输出参数：turbinId, date, avgWindSpeed, dayProduction

(6) 点击“下一步”按钮。

步骤2 选择分析数据

(1) 选择要分析的设备类型：windfarm_LQZNw_001

(2) 选择要分析的字段：ts、turbinId、windSpeed

(3) 选择数据时间段：2016/10/1 ～ 2016/10/15

(4) 点击“下一步”按钮。

步骤3 选择分组字段

(1) 选择分组字段：每个设备数据分为一组。

(2) 点击“保存”按钮结束创建向导。

三、运行分析项目

(1) 选中要运行的分析项目，点击“运行”按钮。（如果弹出对话框询问作业名称，可以留空）

四、查看分析结果

在左侧导航栏里点击“分析作业”可以查看每个分析程序的运行状态和运行结果。

(1) 点击“文件”链接可以看到分析应用输出的原始文件列表。

(2) 点击“接口”链接可以看到json格式的输出，方便基于分析结果开发应用程序。

(3) 点击“浏览”链接可以预览分析应用输出的结果内容。
